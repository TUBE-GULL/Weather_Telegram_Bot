<h2 align="center">Trequirements</h2>

## Installation

````bash
# install package
$ pip install aiogram
````
## Running the app

```bash
# start in linux and mac
$ python3 main.py

# start in windows 
$ python main.py
```

```bash
1 /start

2 /city and /coordinates

3 /help

4 /weather

5 /days and /weeks

6 /stop
```
